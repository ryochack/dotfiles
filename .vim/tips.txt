" === How to Manage Vim Plugins ===
pathogen.vim is a Vim Plugin which Manage Vim Plugins.
pathogen is enable to many Vim Plugins install .vim/bundle Directory.


" === How to Make Vim Document === "
$ cd /usr/local/doc/
$ svn export http://cvs.kaoriya.net/svn/vimdoc/trunk vimdoc
$ cd vimdoc
$ make
$ cp build/runtime/doc/* ~/.vim/doc/
